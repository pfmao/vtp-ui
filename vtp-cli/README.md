# vtp-cli

vtp-ui 组件库的专属命令行工具.

## Install

**npm**

``` bash
cd ./vtp-cli
npm link
```

## Options

**-V, --version**

> 输出 CLI 工具的版本号

**-h, --help**

> 输出使用帮助

## Commands

**new**     

> 新建一个普通组件或者业务组件.

```bash
vtp new # 运行之后根据提示进一步配置
```

**del**  

> 删除一个普通组件或者业务组件.

```bash
vtp del # 运行之后根据提示进一步配置
```
